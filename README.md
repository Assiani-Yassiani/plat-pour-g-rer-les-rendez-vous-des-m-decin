# Présentation

Ce projet consiste en le développement d’une application web qui permet aux clients d’accéder à une large gamme de services de bricolage et d’assistance : ménage, peinture, travaux de bâtiment, plomberie, et bien d’autres.  

L’objectif de la plateforme est de simplifier les tâches du quotidien en offrant une interface simple, rapide et accessible permettant de trouver facilement un prestataire qualifié.

---

# Fonctionnalités principales

- Consultation des services disponibles  
- Demande de prestation (ménage, peinture, plomberie…)  
- Gestion des comptes clients  
- Interface simple et intuitive  
- Développement backend en **PHP**

---

# Technologies utilisées

- **PHP** (backend)  
- Base de données (MySQL ou autre)  
- HTML / CSS  
- JavaScript  

---

# Objectif du projet

Créer une plateforme pratique et accessible qui facilite la recherche de services de bricolage et d’aide domestique, tout en offrant une bonne expérience utilisateur.
